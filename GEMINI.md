# AI Agent Guidelines (AIエージェント操作ガイドライン)

このファイルは、本プロジェクトにおいてAIエージェント（Gemini / Antigravity）が遵守すべき最優先のルールを定めたものです。

## 必須ルール (Mandatory Rules)

1. **操作範囲の制限**
   - AIエージェントが自由に操作・編集・実行してよい範囲は、**本プロジェクトディレクトリ内**のみとします。プロジェクト外のファイルやシステム設定に対する不要な操作は禁止します。
   - ファイルやフォルダを新規に作成・追加する際、指示がない限り**本プロジェクトディレクトリ内**に配置することを厳守してください。

2. **使用言語**
   - ユーザーへの返答や報告は、**すべて日本語**で行ってください。

3. **作成する文書の言語**
   - 実装計画書（implementation_plan.md）やタスク管理表（task.md）、ウォークスルー（walkthrough.md）など、AIエージェントが作成する各種ドキュメントも**すべて日本語**で記述してください。

4. **実装計画書の作成タイミング**
   - ユーザーから明示的に「実装して」等の指示があるまで、実装計画書（implementation_plan.md）の作成や実装の開始は行わないでください。

5. **タスクの実行方法（バックグラウンド実行の制限）**
   - すぐに終わるタスク（短時間で完了するコマンドなど）は、バックグラウンドではなく、同期的に（フォアグラウンドで）実行してください。

6. **Gitコマンドの実行場所**
   - Gitコマンドを実行する際は、カレントディレクトリ（Cwd）を必ず `tamaya` ディレクトリ（`/Users/matsuzawahiroshi/Desktop/project/tamaya`）に設定してください。親ディレクトリなどの異なるディレクトリで実行すると、不要なファイル走査が発生し実行に非常に時間がかかるため厳禁とします。

7. **コーディングルールの厳守**
   - HTML、CSS、JavaScriptのコードを作成・編集する際は、必ず [.agents/rules/coding.md](file:///Users/matsuzawahiroshi/Desktop/project/tamaya/.agents/rules/coding.md) に定義されたコーディングルールを厳守してください。特に、CSSやJavaScriptはHTML内に記述せず、必ず外部ファイルに分離して記述してください。
